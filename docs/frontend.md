# Frontend

## Responsabilidades
- consultar a API consolidada;
- exibir estados de carregamento, erro e vazio;
- oferecer um fluxo operacional simples para cadastro de empresa e registro de sinais.

## Módulos
- `src/lib/api.js`: client HTTP.
- `src/pages/DashboardPage.jsx`: visão executiva.
- `src/pages/SourcesPage.jsx`: catálogo de fontes.
- `src/pages/CompaniesPage.jsx`: operação ponta a ponta.
