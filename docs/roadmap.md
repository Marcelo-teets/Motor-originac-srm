# Roadmap inicial

## Próximas entregas sugeridas

### 1. Endpoint de propostas
- criar endpoint para recebimento de propostas;
- validar payloads de entrada;
- devolver respostas consistentes para integração futura.

### 2. Regras de análise
- evoluir o serviço de análise;
- introduzir regras de elegibilidade e pré-validação;
- preparar respostas mais ricas para decisão inicial.

### 3. Persistência
- definir estratégia de banco de dados;
- mapear entidades principais;
- armazenar propostas e resultados de análise.

### 4. Autenticação e autorização
- proteger endpoints;
- definir perfis de acesso;
- preparar integração com identidade corporativa, se necessário.

### 5. Observabilidade
- estruturar logs de aplicação;
- adicionar métricas básicas;
- preparar tracing e monitoramento.

### 6. Integrações externas
- integrar com fontes de dados e serviços terceiros;
- tratar timeouts, falhas e contratos externos;
- adicionar camada de adaptação para reduzir acoplamento.
