# Paperclip Agent Topology

## Papel dentro do Motor
Esta pasta descreve como o Paperclip deve operar como camada de orquestracao para os agentes do projeto.

## Topologia sugerida
- CEO Orchestrator
- Collection Motor
- Enrichment Supervisor
- Qualification Agent
- Pattern Analyst
- Commercial Board

## Principio operacional
Os agentes Paperclip nao sao a fonte de verdade do projeto.
Eles apenas coordenam chamadas para o backend do Motor e devolvem trabalho explicavel para o time comercial.
