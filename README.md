# motor-originacao

Frontend completo em React + Vite para um motor de originação com páginas de login, dashboard, nova proposta, clientes e pipeline.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Estrutura

- `src/components`: componentes reutilizáveis
- `src/pages`: páginas da aplicação
- `src/services`: dados mockados e camada preparada para serviços futuros
- `src/styles`: estilos globais
- `src/router`: configuração de rotas

## Deploy

O projeto está pronto para deploy na Vercel. O arquivo `vercel.json` faz o rewrite necessário para funcionar como SPA com React Router.
