# Motor Originação SRM

Frontend em **React + Vite + JavaScript** para uma plataforma de originação com aparência de produto SaaS, pronto para deploy inicial na Vercel.

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Estrutura

- `src/components`: componentes reutilizáveis de UI e layout
- `src/pages`: telas da aplicação
- `src/services`: camada mockada pronta para futura integração com backend
- `src/data`: mocks centralizados
- `src/routes`: configuração do React Router
- `src/styles`: estilos globais da aplicação
